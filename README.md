# City Life Economy Sim

A top-down sandbox simulation game with finite resources and a live economy chain.

## Features

- Dynamic city simulation with population management
- Live economy system with resource production and trading
- Interactive canvas-based gameplay
- Save/Load game functionality
- Configurable world settings
- Real-time statistics and market tracking

## Installation

```bash
git clone https://github.com/3eqp/world-game.git
cd world-game
npm start
```

## Usage

1. Open your browser and navigate to http://localhost:8080
2. Click "Add Person" to start building your city
3. Manage resources, professions, and watch your economy grow

## Requirements

- Node.js 18.0.0 or higher
- Modern web browser with Canvas support
